Dotfiles
========================

Dotfiles uses `Dotbot <https://github.com/anishathalye/dotbot>`_ for installation.


Making Local Customizations
---------------------------
- ``git`` : ``~/.gitconfig.local``
- ``pwsh`` : ``~/local.psm1```
- ``vim``: ``~/local.vim``
- ``zsh``: ``~/.zshrc.local_{before,after}``
