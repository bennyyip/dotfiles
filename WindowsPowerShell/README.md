# “That” fancy PowerShell prompt

To use:

- Install PSCore;
- Install lua(http://luabinaries.sourceforge.net/)
-  `ni ~\Documents\WindowsPowerShell\ -i SymbolicLink -ta ~\dotfiles\WindowsPowerShell`
- Install / update modules:
  - `posh-git`
  - `PSReadline`
  - `Get-ChildItemColor`
  - `cd-extras`
